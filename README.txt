SafeHer – Resources & Support Directory

Developers: Abimbola (Lead – Resources Page), Favour, Namakau
Group: CodeCrafters
Date: 2025

Overview

The Resources & Support Directory page is designed to help girls and women in Nigeria quickly find verified support services during situations involving domestic violence, sexual assault, mental health crises, or human rights violations.

The page is intentionally simple, structured, and easy to navigate. Each section contains trusted organisations with clear contact details such as phone numbers, emails, and websites.


What This Page Includes

1. Page Header & Introduction

A clear title section and an introductory message that explains the purpose of the directory and guides visitors on how to use the page.

2. Category Navigation Buttons

Three quick-jump buttons:

NGOs & Support Centres

Mental Health Support

Legal Aid & Human Rights

These help users skip directly to the section they need.

3. Resource Cards (12 in total)

Each card includes:

Organisation name + FontAwesome icon

Short description

Phone lines

Email

Physical location

Website link (opens in new tab)

Cards are displayed in a grid layout:

Two cards per row on desktop

One card per row on mobile

4. Emergency Box

A standout alert box using the darker shade of the brand’s primary color.
It tells users what to do if they are in immediate danger.

5. Quick Exit Button (Safety Feature)

A floating button that stays visible while scrolling.
When clicked, it instantly redirects the user to the homepage for privacy and safety.

6. Quick Call Floating Button

A fixed emergency call button at the bottom-right corner of the page.
Perfect for mobile users — tapping it opens the hotline directly.
